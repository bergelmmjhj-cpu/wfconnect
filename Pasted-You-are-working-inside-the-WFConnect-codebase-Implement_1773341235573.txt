SYSTEM TIMEZONE STANDARD

The entire system must operate in Toronto time.

Timezone:
America/Toronto

Never rely on server timezone.

Create a centralized time utility:

/server/utils/time.ts

Functions:
nowToronto()
toToronto(date)
formatToronto(date)

Use Luxon:

import { DateTime } from "luxon"

export function nowToronto() {
  return DateTime.now().setZone("America/Toronto")
}

export function toToronto(date) {
  return DateTime.fromJSDate(new Date(date)).setZone("America/Toronto")
}

export function formatToronto(date) {
  return toToronto(date).toFormat("yyyy-LL-dd HH:mm")
}

All scheduling, reminders, SMS, logs, CRM display, and mobile app display must use this utility.
