CONTEXT / RULES
- HR is the only role that assigns workers.
- Clients submit shift requests; if no specific worker requested, we broadcast to eligible workers.
- Workers accept/decline.
- Push notifications must exist for shift requests/offers and messages.
- We also want automated reminders (“on my way”, check-in/out, etc.).

AUTOMATION IDEAS TO INCLUDE (at minimum)
Scheduling & Assignment
- Auto-suggest best workers (ranked list) based on: distance, reliability score, cancellations/no-shows, skill match, recent workload fairness, client preference.
- Auto-broadcast in waves: send to top 20 first; if not filled in 10 minutes, expand to next 50; then all eligible.
- Auto-fill recurring requests: clients can create repeating requests; HR can one-tap “approve recurring + auto-broadcast each occurrence.”
- Auto-hold & escalation: if assigned worker hasn’t confirmed by T-12h, auto-broadcast backup offers.
- Auto-backup list: always keep 1–2 “standby” workers offered as backup.

Messaging & Communications
- Auto-message templates triggered by events (request submitted, worker assigned, worker declined, shift cancelled).
- Smart nudges: if client sends request missing key info, prompt client to complete fields before submission.
- Auto-translate optional toggle for worker/client (EN/FR) for key messages.

Attendance & Timesheets
- Geofenced check-in (if feasible) OR “location confirmation” with workplace address.
- Auto-time capture: check-in/out creates timesheet entry; HR only approves exceptions.
- Auto-flag anomalies: late check-in, early leave, no check-in, overlapping shifts.

Payroll & Invoicing
- Auto-calc hours from check-in/out, apply break rules, overtime rules (configurable).
- Auto-generate payroll preview for HR approval each cutoff.
- Auto-generate client invoice from approved shifts; send PDF/email (if email service exists).

Compliance & Onboarding
- Auto-remind workers to complete onboarding docs before their first shift.
- Block assignment if required docs missing; allow HR override with warning + audit log.
- Auto-expiring certifications with reminders.

Quality Control & Issue Handling
- Post-shift micro-feedback: client rates shift + notes; worker reports issues.
- Auto-create incident ticket when “I have an issue” is tapped; notify HR immediately.
- Auto-blacklist / warning escalation when repeated issues occur (configurable thresholds).

Admin & Reporting
- Daily digest for HR: unfilled shifts, upcoming shifts, late check-ins, message backlog.
- Client dashboard: upcoming shifts, assigned workers, last-minute replacements.
- Worker dashboard: next shifts, required actions (confirm, on-my-way, check-in).
